# Architecture Decision Records

Thư mục này lưu các quyết định kiến trúc có ảnh hưởng dài hạn đến project.

ADR khác với tài liệu milestone:

- Milestone mô tả cách triển khai một bước cụ thể.
- ADR ghi lại quyết định kiến trúc, bối cảnh, lựa chọn bị loại, hệ quả và nguyên tắc áp dụng về sau.
