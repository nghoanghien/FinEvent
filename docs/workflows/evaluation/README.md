# Evaluation Workflow Docs

Nhóm tài liệu này mô tả cách đánh giá end-to-end cho FinEvent-VN.

| File | Vai trò |
| --- | --- |
| [evaluation-workflow.md](evaluation-workflow.md) | Workflow đánh giá metrics, error analysis, hallucination metrics và ablation |
| [academic-figures.md](academic-figures.md) | Bộ biểu đồ học thuật bằng `pandas`, `matplotlib`, `seaborn` cho báo cáo và slide |

Khi chạy evaluation thực tế, nên đọc theo thứ tự:

1. `evaluation-workflow.md` để hiểu input/output và metric.
2. `academic-figures.md` để biết các biểu đồ cần xem sau khi chạy xong.
3. `reports/evaluation/report_index.md` để mở đúng artifact mới nhất.
